Para criar uma classe Java ou um pacote no intellij usamos o atalho alt + insert, digitamos esse atalho depois de selecionar o diretório do projeto que estamos trabalhando.

O atalho para executar e compilar arquivos Java no intellij é ctrl + shift + f10

Depois da primeira vez que executamos ctrl + shift + f10 basta apertar a partir de agora shift + f10.

No **eclipse** para comentar todo um bloco de uma vez selecionamos o bloco de código e digitamos **ctrl + shift + c**

Para comentar um bloco de código no Intellij usamos o comando ctrl + shift + /, para descomentar basta usar o mesmo comando. Podemos usar apenas o **ctrl + /** para fazer comentário, mas dessa forma ele comenta linha por linha.  

O atalho **ctrl + alt + o** é para apagar imports que não estamos utilizando em nosso código.

O comando para excluir linhas no intellij é **ctrl + y**.

Fim da aula
